Plaid 是一款遵循 Material 风格设计的 Android App、 聚合了主流创意&设计相关资讯的新闻应用，想知道一款完整的 Material Design 的 App 应该怎么做？就看看它吧。

Cheesesquare Sample
===================================

Demos the new Android Design library. This is not an exhaustive sample, but shows
some of the important features in the Design library:

- Collapsing Toolbar
- FloatingActionButton
- View anchoring
- NavigationView
- Snackbar

Pre-requisites
--------------

- Android SDK v22
- Android Build Tools v22.0.1
- Android Support Repository r16 (for v22.2.1)
